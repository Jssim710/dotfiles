# Vim Configuration & Plugins

This directory contains the Vim configuration and a setup script to automate the installation of `vim-plug` and various productivity plugins.

## Setup
Run the following command from the root of the repository to set up your Vim environment:
```bash
bash 2-vim/vim-set.sh
```

## Installed Plugins & Usage
Please refer to the following blog post for detailed information on the installed plugins and their usage:
[https://blog.jssim710.workers.dev/dotfiles/vim/plugin/](https://blog.jssim710.workers.dev/dotfiles/vim/plugin/)

## Key Mappings
- `<Ctrl-b>`: Re-mapped to `<Ctrl-w>` for window navigation (consistent with tmux prefix).
- `<Space>`: Used as the leader key for `easymotion`.
