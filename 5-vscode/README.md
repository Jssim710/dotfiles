# VS Code Configuration

This directory contains configuration and settings for VS Code.

## Shortcuts
| Shortcut | Action |
| :--- | :--- |
| `` Ctrl + Shift + ` `` | Launch terminal |
| `Ctrl + Shift + g` | Source Control |
| `Ctrl + Shift + x` | Extensions |
| `Ctrl + Shift + w` | Close window |
| `Ctrl + w` | Close tab |
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + b` | Toggle side bar (left) |
| `Ctrl + j` | Toggle panel (bottom) |
| `Ctrl + k` -> `z` | Zen mode |
| `Ctrl + t` | Launch new empty text file (custom) |

> **Note:** The terminal is configured to open as an editor tab by default (`"terminal.integrated.defaultLocation": "editor"` in `settings.json`).

## Tips
### Change Language
1. Press `Ctrl + Shift + P` to enter the Command Palette.
2. Search for `Configure Display Language` and select your preferred language.
